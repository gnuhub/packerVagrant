IBM® Security Identity Manager Adapter README

Adapter Installation

The adapter package included in this archive is an encrypted package that must be decrypted in order to proceed with the adapter installation. Decrypting the adapter package requires IBM Security Identity Manager Adapter Installer. The IBM Security Identity Manager Adapter Installer is used to decrypt all adapter packages

Please download the IBM Security Identity Manager Adapter Installer from IBM Passport Advantage and refer to accompanying documentation for the instructions. Once the adapter package has been decrypted, follow the installation instruction included in the adapter package.
